    This is an ongoing project I've been working on this summer with the goal of creating a "dictionary" that documents a Chinese-language learner's progress, and how it connects to the characters and words they have previously learned. This project is being done in a combination of Python apps I built to make datapoint input easier, as well as a three.js-based javascript library that makes the files created by the python apps into networks of Chinese characters.

    What I learned:

    String manipulation in Python, JSON formatting, three.js

    This website was built with love by Sarah Langleben in 2021 :)
