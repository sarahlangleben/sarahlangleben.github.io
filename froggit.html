    Froggit was a final "exam" for my very first "real" programming course at Cornell—  CS 1110: Introduction to Computing Using Python. This was also my first time working on such a large programming project. Tasks included creating a welcome screen, loading level files, moving the frog based on user input from the keyboard, moving obstacles, and winning/losing the game.

    What I learned:

    OOP with Python, debugging, coroutines, programming an open-ended project that is not fully specified.
